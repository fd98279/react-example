# react-example

React example 

To run the app locally:

```bash
git clone https://github.com/mzabriskie/react-example.git
cd react-example
To run backend
npm run start:backend
http://localhost:3000/users
To run the UI
npm run setup
npm run dev
open http://localhost:8080
To run the mocha test runner
npm run "test:watch"
To run the es-lint
npm run "lint:watch"
```
